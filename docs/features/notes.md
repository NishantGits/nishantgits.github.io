# 📝 Notes & Knowledge Base ([Paid Plan](/features/paid-plan))

FlowChat's Notes module provides a powerful knowledge management system integrated directly with your AI conversations.

## What is the Notes Module?

Notes are rich Markdown documents that let you:
- Capture and organize information from AI chats
- Create reusable knowledge resources
- Link notes to specific conversations
- Build a personal or team knowledge base
- Search across all your saved content

## Creating Notes

### From Scratch
1. Navigate to **Sidebar → Notes**
2. Click **+ New Note**
3. Add title and content in Markdown
4. Save and organize

### From Chat Conversations
1. Find valuable information in any chat
2. Click the save/clip button
3. Select or create a note
4. The excerpt is automatically added

### Using Quick Capture
While chatting, quickly save insights:
- Select text in any message
- Choose "Save to Note"
- The content is captured instantly

## Note Features

### Rich Markdown Support
- **Text formatting** - Bold, italic, headings, lists
- **Code blocks** - Syntax highlighted snippets
- **Tables** - Organized data presentation
- **Links** - Internal and external linking
- **Images** - Attachments and embeds

### Tagging System
- Add custom tags to notes
- Filter notes by tag
- Build topic collections
- Color-coded organization

### Linking to Chats
- Connect notes to specific conversations
- Reference chat context in notes
- Navigate between related content
- Maintain conversation lineage

### Attachments
- Upload files to notes
- Link to external resources
- Embed media content
- Code snippet storage

## Full-Text Search

### Search Capabilities
- Search across all notes
- Search within specific notes
- Cross-reference with chat history
- Filter by date, tag, or content type

### Search Syntax
- **Keywords** - Find specific terms
- **Phrases** - Search for exact matches
- **Tags** - Filter by #tag
- **Dates** - Find content by creation time

## Integration with Conversations

### Injecting Notes into Chats
Use notes as context:
1. Start a new conversation
2. Reference a note in your prompt
3. The AI has access to that knowledge
4. Responses are informed by your saved content

### Building on Past Work
- Reference previous notes in new chats
- Iterate on ideas across multiple sessions
- Maintain continuity over time
- Build complex knowledge structures

## Collaboration Features

### Sharing Notes
- Share with specific team members
- Make notes available organization-wide
- Read-only sharing for reference material
- Collaborative editing for team documents

### Version History
- Track changes over time
- Revert to previous versions
- See who made what changes
- Audit trail for important documents

## Best Practices

1. **Capture immediately** - Save valuable insights while chatting
2. **Tag consistently** - Use a clear tagging system
3. **Link liberally** - Connect related notes and chats
4. **Review regularly** - Clean up and organize periodically
5. **Share knowledge** - Make useful notes available to your team

## Comparison: Notes vs Chat History

| Aspect | Chat History | Notes |
|--------|--------------|-------|
| **Structure** | Linear conversation | Organized document |
| **Search** | Full-text | Full-text + tags |
| **Editing** | Fixed | Fully editable |
| **Sharing** | Per conversation | Granular control |
| **Purpose** | Transient discussion | Persistent knowledge |
| **Context** | Single session | Multiple sessions |
