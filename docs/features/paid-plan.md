## FAQ

### Can I switch between plans?
Yes, you can upgrade or downgrade your plan at any time. Changes take effect immediately.

### What happens to my data if I cancel?
Your data remains accessible in read-only mode. You can export all your content before the plan expires.

### Are there any hidden fees?
No - all pricing is transparent. You only pay for what's listed in your plan.

### Can I use my own API keys?
Yes, paid plan users can connect custom API endpoints alongside FlowChat's built-in models.
